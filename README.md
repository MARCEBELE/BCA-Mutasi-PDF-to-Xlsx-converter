# BCA Mutasi PDF to Excel Converter

Ribet nga sih, kalau mau ngolah data mutasi BCA karena bentuknya PDF

Harus buka PDF satu satu terus copas manual satu-satu ke Excel.

**Program ini solusinya!** Unduh versi terkini di [**releases**](https://github.com/MARCEBELE/BCA-Mutasi-PDF-to-Xlsx-converter/releases).

---

## Cara Pakai

Tinggal jalanin programnya, dan Search folder atau file Mutasi BCA lalu Klik Convert ALL


Excel nya langsung jadi, mudah kann
